# Introduction

## Overview
Welcome to the introductory player guide for Brits PvE Worlds rust server, doesn’t matter if you are an inexperienced newbie or a veteran in this wide community, this wiki is made for everyone alike !!

The main scope of this file is to help people understand parts that they might have not understood, give advice and also help others understand the beauty of this server but I think this is enough of an introduction of the server, and it's best if we keep it short and walk right through the information that I need to share, there is a lot to cover after all and it's all ever-changing!

## What this server offers
- PvE-focused gameplay (minimal PvP elements).
- RPG-style progression systems.
- Custom items & gear tiers.
- Boss fights & special events.
- Player driven economy & trading.
- Long-term progression across wipes.

ATTENTION: This guide is community-made and constantly updated as the server evolves. Some mechanics may change over time.
